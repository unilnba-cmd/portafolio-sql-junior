# Portfolio Architecture

This repository is structured to simulate a real-world SQL analytics environment.

## Data Layer

The project includes simulated datasets representing a retail business.

Datasets include:

- customers
- products
- sales transactions

These datasets are used to perform analytical queries and business insights.

## Analytics Layer

Business analysis queries are organized in:

datasets/08_business_questions

These queries answer common business questions such as:

- revenue analysis
- top customers
- revenue ranking
- revenue by category
- monthly revenue trends

## SQL Skills Demonstrated

The project demonstrates the use of:

- JOIN operations
- GROUP BY aggregations
- window functions (RANK)
- revenue calculations
- time-based analysis

## Objective

The goal of this repository is to demonstrate practical SQL analytics skills through structured business questions and data analysis workflows.
